__Description__ 
 : X <is beneficial for> Y. Agent X behaves in a way that produces some result Y that is advantageous to some beneficiary. A means X through which a consequence Y can be achieved. E.g. "biological control arthropods" <is beneficial for> "biological control"; "pest control" <is beneficial for> "plant health";
